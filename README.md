# Vue 3 + TypeScript + Vite

## Node 版本需求

此專案已升級到 **Vite 7**，需要 **Node 20.19+**（或 **Node 22.12+**）。

如果你使用 `nvm`：

```bash
nvm install 20.19.0
nvm use 20.19.0
```

## 安裝與開發

```bash
cd /Users/gonsakon/Documents/github/2025-backend/2025-backend-camp-game/frontend
npm install
npm run dev
```

## 建置

```bash
npm run build
```

> 備註：專案已提供 `.npmrc`，會把 npm cache 放在專案內的 `.npm-cache`，可避免 `~/.npm` 權限問題。

Learn more about the recommended Project Setup and IDE Support in the [Vue Docs TypeScript Guide](https://vuejs.org/guide/typescript/overview.html#project-setup).
